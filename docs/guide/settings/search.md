# 搜索设置

控制在线搜索和批量匹配的行为。

## 插件管理

进入路径：`设置` → `搜索设置` → `插件管理`。

管理已安装的搜索源插件，包括导入、启用/停用、排序、配置和卸载。完整说明见 [使用插件](../plugins.md)。

## 单次加载数量

设置每个插件源每次加载的搜索结果数量。可通过结果列表底部的“加载更多”按钮继续加载；数值越大，每次加载的候选项越多，但可能消耗更多流量，也会增加图片缓存。

## 优先使用文件名匹配

影响批量匹配的行为：

- **关闭**（默认）：使用音频文件内置的标题和艺术家标签作为搜索关键词。
- **开启**：忽略内置标签，优先用文件名搜索。适合文件标签混乱但文件名较规范的音乐库。

启用此选项时，建议将标题和艺术家字段的写入规则设置为 **"覆盖"** 模式，以确保匹配结果能正确写入标签。
